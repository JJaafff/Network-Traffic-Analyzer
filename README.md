Capturing and Analyzing traffics, organizing them well, so that it is easier to monitor.
Firstly install requirements by typing pip install requirements.txt
after that you should run the file with the following command: streamlit run dashboard.py (don't forget cmd or Powershell has to be in Administrator mode, other way the it will not be able to capture any packet)
then you will see the web page on your default browser, you should choose the interface and everything is okay, your packets will appear and be organized, nowhere saved, only for you, deleted after Ctrl+c in cmd or powershell.
